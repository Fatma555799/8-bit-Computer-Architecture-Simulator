# 8-bit Computer Architecture Simulator

## 📋 Project Description

This is an educational **8-bit computer simulator** built on CircuitVerse to demonstrate the basic principles of computer architecture and digital logic design.

The project simulates a simple CPU with visible data paths, control signals, and component interactions. It helps users understand how instructions are fetched, decoded, and executed in a basic processor.


## 🛠️ Features

- 8-bit CPU architecture
- Arithmetic Logic Unit (ALU)
- Registers and Program Counter
- Memory simulation (RAM/ROM)
- Instruction decoder and Control Unit (CU)
- Clock and manual stepper for step-by-step execution
- Visual representation of data buses and signal flow

## 📁 Main Components

- Half Adder & Full Adder
- Decoders (3×8, 4×16, 2×4)
- Registers
- ALU
- Memory cells
- Clock & Stepper
- SHFL, CU, and supporting logic blocks

## 🚀 How to Run the Simulation

1. Open the project using this link:  
   [View 8-bit Computer Simulator](https://circuitverse.org/users/283483/projects/computer-d9e29221-1281-4397-9f6b-c2693dbde11a)

2. Click **"Open in Simulator"** or **"Launch Simulator"**.

3. Use the **Clock** button for automatic ticking or the **Stepper** for manual step-by-step execution.

4. Observe how data moves between registers, ALU, memory, and the control unit.

## 🎯 Learning Objectives

- Understand the basic structure of a CPU
- See the interaction between ALU, registers, control unit, and memory
- Learn how simple instructions are executed in a digital computer
- Visualize data flow and control signals
